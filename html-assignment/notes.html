Some Tag Descriptions

img:
It is used to embed images in a web page

ul:
Unordered list. It is used to create a bulleted list.

Blockquote:
It is used to define a block of text that is a quotation from another source.

p:
Stands for paragraph. It is used to define paragraphs in HTML. It represents a block of text.

a
Anchor. It is used to create hyperlinks. It can link to other web pages, files, or specific secitons within a page.
